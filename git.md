Git
===

Intro
-----

### 개념
  * 로컬저장소
  * 작업트리
  * 스테이징
  * 원격저장소

기본명령어 
---------

### 환경설정

>$ git config --global --list 

### 저장소 생성

>$ git init 

### Indexing 

>$ git add finaname 

### Commit 

>$ git commit -m "commit message" 


원격저장소
---------

### 원격저장소 추가

>$ git remote add origin 저장소

### 푸시

>$ git push origin master
