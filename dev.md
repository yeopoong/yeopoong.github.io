Jenkins 
-------

http://jenkins-ci.org/

```
# adduser jenkins
# su - jenkins
# wget http://mirrors.jenkins-ci.org/war-stable/latest/jenkins.war
# java -jar jenkins.war --httpPort=5000
```

`http://호스트:5000/`로 접근해서 확인해 보자!
