## DFS(Depth First Search)
> 깊이 우선 탐색

연결된 노드그룹

```java
// 연결된 모든 노드를 방문처리 한다.
int dfs(int c, List<Integer>[] graph, boolean[] visited) {
    if (visited[c]) return 0;
    
    visited[c] = true;
    
    for (int v : graph[c]) {
        dfs(v, graph, visited);
    }
    
    return 1;
}
```
