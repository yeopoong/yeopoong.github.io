Compile & Execute
-----------------

`Hello.scala`
```scala
object Hello {
  def main(args: Array[String]) = println("Hello World!")
}
```

```
C:\dev\scala\hello>scalac Hello.scala
Hello$.class   Hello.class    Hello.scala

C:\dev\scala\hello>scala -classpath . Hello
Hello World!
```
